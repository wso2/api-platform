# WSO2 Gateway Policy SDK

Python SDK for implementing WSO2 API Platform gateway policies.

Recommended imports use the package root:

```python
from wso2_gateway_policy_sdk import RequestPolicy, ProcessingMode
```

If you need to pin a specific SDK contract version explicitly, use:

```python
from wso2_gateway_policy_sdk.policy.v1alpha2 import RequestPolicy, ProcessingMode
```
