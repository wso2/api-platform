# API Platform Command Line Interface Tool

## Development

### Prerequisites

- Go 1.25.x
- Make

### Building from source

```bash
# Generate the ctl executable for your OS
make build
```

```bash
# Generate executables for Linux, MacOS, and Windows in both amd64 and arm64
make build-all
```

```bash
# Run tests
make test
```

```bash
# Clean the build directory
make clean
```
