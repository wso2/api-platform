# Petstore API FAQ

## What is the Petstore API?

The Petstore API is a sample Swagger API used for examples, testing, and API portal demos.

## How do I use the `api_key` header?

Add `api_key` as a header with your API key value:

`api_key: special-key`

## What are the main resources?

- `pet`
- `store`
- `user`
