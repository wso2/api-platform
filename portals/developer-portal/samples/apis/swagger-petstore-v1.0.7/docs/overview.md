# Swagger Petstore API Overview

The Swagger Petstore API is a sample REST API for managing pets, orders, and users. It is based on the OpenAPI Petstore example and demonstrates common API features:

- CRUD operations for pets
- Order management for pet purchases
- User registration, login, and profile management

## API Base URL

`https://petstore.swagger.io/v2`

## Key resources

- `/pet`
- `/store/order`
- `/user`
